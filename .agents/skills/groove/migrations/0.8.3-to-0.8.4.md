# Migration: 0.8.3 → 0.8.4

## Outcome

`AGENTS.md` sections are rewritten to the v0.8.x format: a 2-line `groove:prime` bootstrap and a 2-line `groove:task` stub.

## Acceptance Criteria

- `<!-- groove:prime:start -->` section contains only the session bootstrap (2 lines)
- `<!-- groove:task:start -->` section contains only the minimal beans stub (2 lines), if `tasks: beans`
- All other `AGENTS.md` content is preserved

## Constraints

- This migration is idempotent — safe to re-run if interrupted
- If `AGENTS.md` does not exist, skip

### Step 1 — Rewrite `groove:prime` section

Replace the entire `<!-- groove:prime:start -->` / `<!-- groove:prime:end -->` block with:

```
<!-- groove:prime:start -->
**Run at the start of every session:**
`/groove:prime` — loads groove workflow context into the conversation
<!-- groove:prime:end -->
```

If the section is absent, skip.

### Step 2 — Rewrite `groove:task` section

Replace the entire `<!-- groove:task:start -->` / `<!-- groove:task:end -->` block with:

```
<!-- groove:task:start -->
Task backend: beans — use `/groove:task` commands for all task management.
Run `beans prime` if you need the full beans CLI reference.
<!-- groove:task:end -->
```

If the section is absent, or `tasks: none`, skip.
